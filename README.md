###Git Commands: 
- git add .                    : adds all
- git add -u                   : adds modified / deleted
- git add --ignore-removal .   : adds modified / new

###Code Formatting 
This is done using one or three backticks "`"
```
int x;
x = 2 + 2;
destroy(x);
```
The above function `destroy(x)` does nothing.
